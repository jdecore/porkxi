# Plan: Separación completa Astro (estático) + Vue (animado/interactivo)

## Objetivo
Separación estricta:
✅ **Astro**: Todo contenido estático, texto, imágenes, estructura, SEO
✅ **Vue 3**: TODO lo que tenga animación, interacción, estado dinámico
✅ **CSS 100%**: Todo diseño, estilos, transiciones visuales
❌ **Eliminar TODO Javascript externo, inline, o cualquier código JS que no sea runtime de Vue y componentes autorizados**

---

## Pasos de implementación

### 1. Limpieza de Javascript existente
- [ ] Buscar y eliminar TODO script tag inline en archivos .astro
- [ ] Eliminar cualquier import de JS externo en layouts/pages
- [ ] Remover eventos onclick, onmouseover, onload inline en HTML
- [ ] Verificar que no quede ningún JS ejecutandose fuera de componentes Vue

### 2. Migración de animaciones existentes a Vue
Todos los elementos animados pasan a ser componentes Vue con `client:visible`:

| Elemento | Ubicación actual | Acción |
|----------|------------------|--------|
| Contadores KPI | `TarjetasKpiAnimadas.vue` | Integrar en la página, habilitar count-up nativo Vue |
| Tooltips hover gráfica | `GraficaInteractiva.vue` | Mantener, optimizar eventos mouse/touch |
| Tabs comparativa Colombia/USA | `TabsComparativa.vue` | Mantener transiciones fade Vue |
| Animación líneas gráfica | `GraficaInteractiva.vue` | Mantener animación SVG por medio de bindings Vue |
| Filtros rango año | `GraficaInteractiva.vue` | Mantener estado interno Vue |

### 3. Verificación separación Astro/Vue
✅ **Archivos .astro contendran SOLAMENTE**:
  - Estructura HTML semántica
  - Contenido texto
  - Importe de estilos CSS
  - Importe de componentes Vue con directiva `client:visible`
  - NINGÚN script, NINGÚN evento inline, NINGÚN lógica

✅ **Componentes Vue contendran SOLAMENTE**:
  - Estado reactivo
  - Manejadores de eventos
  - Animaciones programadas
  - Interacción usuario
  - NINGÚN estilo inline (todo pasa a archivos CSS)

✅ **Archivos CSS contendran SOLAMENTE**:
  - Todo diseño visual
  - Colores, fuentes, espaciados
  - Transiciones CSS
  - Media queries responsive
  - NINGÚN lógica, NINGÚN `@keyframes` controlado por JS

### 4. Verificación final build
- [ ] Ejecutar `npm run build`
- [ ] Validar que en `dist/` solo existe:
  - Archivos HTML estático
  - Archivos CSS
  - Único bundle JS: runtime Vue + los 3 componentes autorizados
- [ ] Comprobar tamaño total dist < 200KB
- [ ] Verificar que no haya ningún otro archivo JS

---

## Criterios de aceptación
1. Todo elemento que se mueva, cambie, reaccione al usuario es un componente Vue
2. No existe ni una sola línea de Javascript fuera de los componentes Vue
3. Todo el diseño visual está exclusivamente en archivos CSS
4. Todo contenido estático, texto y estructura está en archivos Astro
5. La hidratación Vue ocurre solamente cuando el usuario scrollea hasta el componente
6. Ningún javascript se ejecuta en la carga inicial de la página
