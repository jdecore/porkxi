# porkξ

Dashboard de periodismo de datos: inventario porcino Colombia vs EE.UU.

Hecho por Juan Enriquez.

## Mensaje central

En Colombia, acceder a datos porcinos oficiales no es sencillo. Los datos están fragmentados, desactualizados o sin API funcional, obligando a usar fuentes periodísticas especializadas en lugar de datos gubernamentales directos.

## Correr localmente

```bash
npm install
npm run dev
```


## Fuentes de datos

- Colombia: Porcinews (enero 2026)
  https://porcinews.com/colombia-el-censo-porcino-registra-mas-de-106-millones-de-animales-en-el-pais/
- EE.UU.: USDA NASS (23 dic 2025)
