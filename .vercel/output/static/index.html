¿Por qué existe esta brecha?

🇨🇴 Colombia

Acceder a información porcina oficial actualizada sigue siendo difícil. Los datos están fragmentados entre ICA, DANE, Porkcolombia e INVIMA.

No existe una API pública y estable equivalente a USDA o Eurostat para la serie consolidada nacional.

En Colombia, acceder a información porcina oficial actualizada es prácticamente imposible por canales directos. Los datos están fragmentados entre varias entidades que no se comunican entre sí. Ninguna fuente gubernamental ofrece una API funcional. Por eso, los datos de esta visualización provienen de Porcinews, un medio periodístico especializado — no de fuentes gubernamentales directas. Eso en sí mismo es el problema.

Por eso, esta visualización usa Porcinews como fuente de referencia para el último dato disponible.

🇪🇺 Europa (UE-27)

Eurostat publica la serie oficial mediante API abierta (`apro_mt_lspig`) con dimensiones estandarizadas.

Esto permite trazabilidad, automatización y comparabilidad entre países sin depender de scraping manual de PDFs.

🇺🇸 EE.UU.

La Livestock Mandatory Reporting Act obliga a reportar al USDA. Hay publicaciones periódicas, verificables y con calendario público.

La disponibilidad trimestral facilita monitoreo continuo y análisis oportuno del sector porcino.
